# CVG Line Maintenance Operations Dashboard

A local-first web application for managing line maintenance operations at Cincinnati/Northern Kentucky International Airport (CVG). Built for Kalitta Air's maintenance team.

The dashboard provides three core views:

- **Flight Board** — Gantt-style timeline showing aircraft on-ground windows, maintenance work packages, and scheduling
- **Statistics Dashboard** — KPI cards, charts, and analytics for operational performance
- **Capacity Modeling** — Demand vs. capacity analysis for staffing decisions across Day, Swing, and Night shifts

## Key Features

- Real-time Gantt chart with zoom, pan, and aircraft filtering
- Configurable shift capacity planning (Day 07-15, Swing 15-23, Night 23-07)
- Role-based access control (user, admin, superadmin)
- 11 theme presets with light/dark/system color mode
- CSV export for tables
- Admin panel for customer colors, aircraft type mappings, data import, and user management
