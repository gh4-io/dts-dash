# Changelog

## Code Review (2026-02-15)

### Security Fixes
- **Secured `/api/seed` endpoint** — removed from proxy exclusions, added superadmin guard in production
- **Removed credential logging** — seed script no longer prints passwords to console
- **Added config key whitelist** — `PUT /api/config` now rejects unknown keys

### Bug Fixes
- **Fixed NaN propagation in `formatDuration()`** — returns `"0:00"` for NaN/Infinity instead of `"NaN:NaN"`
- **Fixed NaN propagation in `formatDurationHuman()`** — returns `"0m"` for invalid input
- **Fixed division by zero in capacity utilization** — returns 0% when `realCapacityMH = 0`

### Code Quality
- **Replaced 17 `console.log` calls** with `console.warn` across data layer for consistent log levels
- **Added ESLint rules** — `no-console`, `no-debugger`, `prefer-const`, `no-var`, `eqeqeq`
- **Added Prettier** for consistent formatting
- **Added eslint-config-prettier** to prevent rule conflicts

### Testing
