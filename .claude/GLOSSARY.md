# Glossary

| Term | Definition |
|------|-----------|
| **CVG** | Cincinnati/Northern Kentucky International Airport (IATA: CVG) |
| **Line Maintenance** | Routine aircraft maintenance performed between flights at the gate/ramp |
| **MH** | Man-Hours — unit of labor measurement |
| **WP** | Work Package — a bundled set of maintenance tasks for an aircraft visit |
| **effectiveMH** | Computed demand MH for a work package after applying inclusion/exclusion rules |
| **Ground Time** | Duration between aircraft arrival and departure at CVG |
| **Operator** | = Customer. The airline whose aircraft is being serviced (CargoJet, Aerologic, etc.) |
| **Registration** | Aircraft tail number (e.g., C-FOIJ, N774CK, G-DHLW) |
| **Fleet Number** | CargoJet's internal numbering: `NNN-RRRR` format (e.g., 506-GTCJ = C-GTCJ) |
| **Aircraft Type** | Model designation: B777, B767, B747, B737 |
| **Station** | Airport code where maintenance occurs. Always "CVG" for this system. |
| **Shift** | Work period: Day (07–15), Swing (15–23), Night (23–07) |
| **Utilization** | (Demand MH / Real Capacity MH) × 100% |
| **OData** | Open Data Protocol — SharePoint's REST API format for our source data |
| **Gantt** | Horizontal bar chart showing tasks/events across a timeline |
| **KPI** | Key Performance Indicator — summary metric displayed as a card |
| **HAR** | HTTP Archive — browser network capture format used for reference analysis |
| **FA** | Font Awesome — icon library, self-hosted |
| **SVAR** | ~~Vendor of the React Gantt component (MIT edition)~~ — replaced by Apache ECharts (D-008) |
